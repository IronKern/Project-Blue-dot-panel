Alle Commands

/bank

Überprüft dein Kontostand.

/bank

/deposit <amount>

Zahlt Geld auf dein Bankkonto ein.

/deposit 100

/withdraw <amount>

Hebt Geld von deinem Bankkonto ab.

/withdraw 50

/gift <user> <amount>

Schenkt einem Benutzer Geld.

/gift @Benutzername 20

/daily

Fordert deine tägliche Belohnung ein.

/daily

/leaderboard

Zeigt die reichsten Benutzer an.

/leaderboard

/shop

Durchstöbert den Shop.

/shop

/steal <user>

Versucht, einen Benutzer auszurauben (riskant!).

/steal @Benutzername

/gamble <amount>

Spielt um Geld - 50/50 Chance.

/gamble 10

/commands

Zeigt dieses Befehlsmenü an.

/commands

/stats

Zeigt deine Statistiken an.

/stats

/love <user>

Berechnet den Liebesprozentsatz.

/love @Partner

/password

Generiert ein sicheres Passwort.

/password

/remind <time> <message>

Setzt eine Erinnerung.

/remind 1h Kaffee trinken

/userinfo

Ruft Informationen über einen Benutzer ab.

/userinfo @Benutzername

/serverinfo

Ruft detaillierte Serverinformationen ab.

/serverinfo

/avatar

Ruft den Avatar eines Benutzers ab.

/avatar @Benutzername

/weather

Ruft Wetterinformationen für eine Stadt ab.

/weather Berlin

/ping

Überprüft die Latenz und Antwortzeit des Bots.

/ping

/math

Berechnet einen mathematischen Ausdruck.

/math 5 * (2 + 3)

/random_number

Generiert eine Zufallszahl zwischen zwei Werten.

/random_number 1 100

/stats (Slash Command)

Zeigt Bot-Statistiken an.

/stats

/owner

Findet heraus, wer der allmächtige Besitzer dieses Bots ist.

/owner

/kick

Kickt ein Mitglied. (Nur für Staff)

/kick @Benutzername Regelverstoß

/timeout

Versetzt ein Mitglied für eine angegebene Minutenanzahl in Timeout.

/timeout @Benutzername 60 Beleidigung

/ban

Verbannt ein Mitglied. (Nur für Staff)

/ban @Benutzername Permanenter Verstoß

/clear

Löscht Nachrichten aus dem Kanal. (Nur für Staff)

/clear 50

/nuke

Erstellt einen Kanal neu. (Nur für Staff)

/nuke

/announce

Erstellt eine Ankündigung. (Nur für Staff)

/announce #ankündigungen Willkommen an alle!

/coinflip

Wirft eine Münze.

/coinflip

/eightball

Stellt dem magischen 8-Ball eine Frage.

/eightball Wird es regnen?

/roll

Wirft einen Würfel mit einer angegebenen Anzahl von Seiten.

/roll 6

/quote

Generiert ein inspirierendes Zitat.

/quote

/choose

Wählt zwischen mehreren Optionen (durch Komma getrennt).

/choose Apfel, Banane, Orange

/timezone <timezone>

Legt deine Zeitzone fest.

/timezone Europe/Berlin

/time

Zeigt deine aktuelle Zeit an.

/time

/time <user>

Zeigt die Zeit von jemandem an.

/time @Benutzername

/timezones

Listet alle verfügbaren Zeitzonen auf.

/timezones

/ticket

Erstellt ein Support-Ticket.

/ticket
